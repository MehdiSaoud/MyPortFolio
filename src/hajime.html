Générateur de devis

Cette application web est un formulaire de saisie d'information pour les personnes qui souhaitent acheter un service de déménagement.

Les données saisie par l'utilisateur sont envoyé à la base MongoDB, un devis est ensuite généré selon les données reçues.

Les maquettes ont été réalisées sur Figma avec l'aide d'un designer web junior.

Pour l'intégration de cette application web nous avons utilisé la stack MEAN (MongoDB | Express | AngularJS | NodeJS).

Nous avons intégrer le stepper et les inputs du formulaire avec la librairie Angular Material.
