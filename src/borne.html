Borne

Borne restaurant

Cette application web est une borne de restaurant qui permet à l'utilisateur de remplir son panier avec différents produits qu'elle propose.

Les produit choisi par l'utilisateur sont envoyé à la base MongoDB, le panier est ensuite rempli selon les données reçues.

Pour l'intégration de cette application web nous avons utilisé la stack MERN (MongoDB | Express | ReactJS | NodeJS).

Le style à été intégré avec le pré-processeur SASS.
